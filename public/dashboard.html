内容预览 — 三步完成多平台适配
  • 第 1 步：输入内容 — 在下方文本框粘贴或输入你要发布的文案（文章、帖子、视频文案均可）
  • 第 2 步：查看预览 — 系统实时生成各平台的适配预览，进度条颜色表示字数：绿色=安全、黄色=接近上限、红色=超出限制
  • 第 3 步：一键复制 — 点击任意平台卡片右上角的 复制 按钮，即可将适配后的内容复制到剪贴板，直接粘贴到对应平台
提示：内容超过10字会自动保存到历史记录；点击"加载示例内容"可快速体验。

工作流使用指南
  • 点击任意工作流卡片展开详情，查看执行步骤和所需参数
  • 展开后点击"运行"按钮可模拟执行工作流（需配置MCP连接后可实际运行）
  • 点击"编辑模式"可修改工作流名称、描述、步骤等（点击卡片进入编辑弹窗）
  • 使用分类标签筛选不同类型的工作流
  • 工作流与 ops-flow MCP Server 联动，连接后支持真实执行
Sales CRM 使用指南
  • 商机漏斗：展示客户从"初次接触"到"已成交"的完整销售路径。每一行的宽度代表该阶段的客户数量，箭头百分比 = 下一阶段数量 / 当前阶段数量 x 100%（即进入下一阶段的转化率）。
  • 线索漏斗：展示潜在客户（线索）按意向强度的分布。从"冷线索"（未接触/无回复）→"温线索"（有互动但未明确意向）→"热线索"（明确表达兴趣）→"已验证"（确认有预算和需求）→"已转化"（升级为商机）→"已流失"（放弃跟进）。
  • 操作方法：点击数字可直接修改数量；点击"编辑模式"后，点击标签名称可编辑阶段详情（名称、颜色），点击 x 可删除，点击 + 可添加新阶段。
  • 颜色选择：编辑阶段时提供12色预设色板，也可手动输入HEX颜色值。
  • MCP同步：连接 sales-crm MCP Server 后，数据可实时同步到后端。

MCP 协议集成 | MCP Protocol Integration

所有工具均支持 Model Context Protocol (MCP)，可直接被 Claude Desktop / Cursor / VS Code 调用。
All toolkits support MCP and can be called directly from Claude Desktop / Cursor / VS Code.

// claude_desktop_config.json
{
  "mcpServers": {
    "media-ops": { "command": "node", "args": ["src/mcp-server/index.js"] },
    "ops-flow": { "command": "node", "args": ["src/mcp-server/index.js"] },
    "sales-crm": { "command": "node", "args": ["src/mcp-server/index.js"] },
    "pm-toolkit": { "command": "node", "args": ["src/mcp-server/index.js"] }
  }
}

完整配置见 mcp-registry.json | 共 73 个 MCP Tools 可用
